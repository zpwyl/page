MarkDowm书写规则

一、标题

使用 # 数量表示标题级别(＃后有空格)

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

二、列表

  • 有序列表

    使用数字点加空格(例如：1. )

  • 无序列表

    使用 *、-、+三个符号

三、区块引用

使用 > 来使用引用

> 一级引用 
>> 二级引用 
>>> 三级引用 
>>>> 四级引用 
>>>>> 五级引用 
>>>>>> 六级引用 
>>>>>>> 七级引用
